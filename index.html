Your First Java Program

Your Progress: Level 3 Beginner

65% complete - Next: Object-Oriented Programming

Let's start with the classic "Hello World" program. This is the first step for every Java programmer! 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

In this code:

  • public class HelloWorld defines a class named HelloWorld
  • public static void main(String[] args) is the main method where program execution begins
  • System.out.println() prints text to the console

Variables and Data Types

Java uses variables to store data. Here are the basic data types:

int age = 25;                    // Integer
double price = 19.99;           // Decimal number
char grade = 'A';               // Single character
boolean isJavaFun = true;       // True or false
String message = "Hello Java";  // Text
