Handling Multiple Values

Lesson 8: Arrays and Strings

Learn how to store multiple values and handle text in Java using arrays and strings!

👋 Welcome back, coding star!

So far, your programs can store single values in variables. But what if you want to store a list of numbers, names, or words?

That's where arrays and strings come in — your tools for handling multiple values and text. 📦✏️

📦 1. Arrays: Your Data Backpack

An array is like a row of boxes, each with a number (index) to keep track of items.

You can store multiple values of the same type in one array.

Syntax:

dataType[] arrayName = new dataType[size];

Example:

int[] scores = new int[5]; // Create an array with 5 boxes
scores[0] = 90;
scores[1] = 85;
scores[2] = 78;
scores[3] = 92;
scores[4] = 88;

System.out.println("First score: " + scores[0]);
System.out.println("Last score: " + scores[4]);

✅ Output:

First score: 90
Last score: 88

💡 Tip: Arrays start counting at 0, not 1!

Shortcut: Initialize Array in One Line

int[] numbers = {2, 4, 6, 8, 10};

Looping Through an Array

String[] fruits = {"Apple", "Banana", "Cherry"};

for (int i = 0; i < fruits.length; i++) {
    System.out.println(fruits[i]);
}

✅ Output:

Apple
Banana
Cherry

✏️ 2. Strings: Handling Text

A String is a sequence of characters, like a sentence, word, or name.

Strings are very powerful in Java.

Example:

String name = "Aaryan";
System.out.println("Hello " + name + "!");

✅ Output:

Hello Aaryan!

Common String Methods

Method What it does Example
.length() Count letters "Hello".length() → 5
.toUpperCase() Convert to uppercase "java".toUpperCase() → "JAVA"
.toLowerCase() Convert to lowercase "JAVA".toLowerCase() → "java"
.charAt(index) Get character at position "Java".charAt(2) → 'v'
.substring(start, end) Extract part of string "Java".substring(1,3) → "av"

🎮 Mini Challenge

Your Challenge

Create an array of your favorite fruits (at least 3). Print each fruit using a loop.

Make a String variable with your full name, then:

  • Print the length of your name.
  • Print it in uppercase and lowercase.

Bonus: Combine arrays and strings: make an array of names and print a greeting for each name.

🌈 Summary

  • Arrays → Store multiple values of the same type, accessible by index
  • Strings → Handle text with powerful methods for manipulation
  • Arrays + Strings = building blocks for more advanced programs

👏 You're now ready to store, manage, and manipulate lots of data in your programs — a huge step toward real-world coding!
