Lesson 4: Operators and Expressions

Learn how to perform calculations and make logical comparisons with operators and expressions in Java!

üëã Hey coder!

You've learned how to store data in variables. But what if you want to add numbers, compare ages, or check if something is true?

That's where operators come in! Think of them as math & logic superpowers for your variables.

‚ûï Math Operators

Just like in school, Java can do math:

Operator Meaning Example (a=10, b=3) Result
+ Add a + b 13
- Subtract a - b 7
* Multiply a * b 30
/ Divide a / b 3
% Remainder (mod) a % b 1

üëâ Example:

int a = 10, b = 3;
System.out.println("a + b = " + (a + b));
System.out.println("a % b = " + (a % b));

Output:

a + b = 13
a % b = 1

ü§î Comparison Operators

These check if something is true or false.

Operator Meaning Example (x=5, y=8) Result
== Equal to x == y false
!= Not equal to x != y true
> Greater than x > y false
< Less than x < y true
>= Greater or equal x >= 5 true
<= Less or equal y <= 8 true

üß† Logical Operators

These combine true/false values (booleans).

Operator Meaning Example Result
&& AND (both true) (x > 0 && y > 0) true
|| OR (either true) (x > 10 || y < 10) true
! NOT (opposite) !(x > y) true

üñ•Ô∏è Example Program

public class OperatorsDemo {
    public static void main(String[] args) {
        int age = 15;
        int score = 80;

        System.out.println("Age next year: " + (age + 1));
        System.out.println("Score doubled: " + (score * 2));
        System.out.println("Is age greater than 18? " + (age > 18));
        System.out.println("Passed exam? " + (score >= 50));
        System.out.println("Teenager? " + (age >= 13 && age <= 19));
    }
}

üëâ Output:

Age next year: 16
Score doubled: 160
Is age greater than 18? false
Passed exam? true
Teenager? true

üåà Summary

  • Math operators help you calculate
  • Comparison operators check conditions
  • Logical operators let you combine true/false values
  • Together, they make expressions that power your programs
