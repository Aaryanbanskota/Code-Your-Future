Storing Information

Lesson 3: Variables, Constants, and Data Types

Learn how to store and use data with variables, constants, and different data types in Java!

👋 Hey future Java pro!

So far, you've made your computer talk. But what if you want it to remember things? Like your name, your age, or even your score in a game?

That's where variables come in. Think of them like labeled boxes where you can keep stuff.

📦 What is a Variable?

A variable is a container that stores data.

You give it a name (label) and a type (what kind of thing it can hold).

int age = 12;
String name = "Aaryan";

Here:

int age = 12; → A box named age that stores the number 12.

String name = "Aaryan"; → A box named name that stores the word Aaryan.

🧰 Common Data Types

Java has different box types for different things:

Type What it stores Example
int Whole numbers 10, -5, 2000
double Decimal numbers 3.14, 99.99
char Single character (a letter) 'A', 'Z', '5'
String Text (words/sentences) "Hello Java!"
boolean True or False true, false

🔒 What about Constants?

Sometimes, you want a value that never changes. That's called a constant in Java.

final double PI = 3.14159;

Here:

final means the value cannot be changed later.

Try changing PI and Java will yell at you 😆.

🖥️ Example Program

public class AboutMe {
    public static void main(String[] args) {
        String name = "Alex";
        int age = 14;
        double height = 5.6;
        boolean student = true;

        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Height: " + height + " ft");
        System.out.println("Student: " + student);
    }
}

👉 Output:

Name: Alex
Age: 14
Height: 5.6 ft
Student: true

🎮 Mini Challenge

Print them in a nice sentence.

🌈 Summary

  • Variables = boxes to store data
  • Different data types store different kinds of values
  • Constants (final) are values that never change

👏 Great job! You've just learned how to make Java remember things.
