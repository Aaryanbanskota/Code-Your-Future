Lesson 10: Classes and Objects in Detail

Learn how to create classes, objects, and constructors in Java with detailed examples!

üëã Welcome back, Java explorer!

You've already met OOP (Object-Oriented Programming). Now, let's open the box and see how Classes and Objects really work.

üß© What is a Class?

A class is like a blueprint üè†.

It doesn't build anything by itself but tells you how to build.

Inside a class, you can define:

  • Fields (variables) ‚Üí to hold data.
  • Methods (functions) ‚Üí to perform actions.

üëâ Think of a Car class. It describes wheels, engine, and drive() behavior, but it's not an actual car.

üöó What is an Object?

An object is an actual thing built from the blueprint.

Example: The "Car" blueprint becomes myCar, yourCar, redCar, etc.

Each object has its own copy of the data.

üñ•Ô∏è Example 1: Defining a Class & Creating an Object

// Define a Car class
class Car {
    // Fields (data/attributes)
    String color;
    String brand;
    
    // Method (action)
    void drive() {
        System.out.println(brand + " car is driving!");
    }
}

// Main class
public class Main {
    public static void main(String[] args) {
        // Create objects (cars)
        Car myCar = new Car();
        myCar.color = "Red";
        myCar.brand = "Tesla";
        
        Car friendCar = new Car();
        friendCar.color = "Blue";
        friendCar.brand = "BMW";
        
        // Call methods
        myCar.drive();
        friendCar.drive();
    }
}

‚úÖ Output:

Tesla car is driving!
BMW car is driving!

üñ•Ô∏è Example 2: Constructor ‚Äì Special Method to Build Objects

class Student {
    String name;
    int age;

    // Constructor
    Student(String n, int a) {
        name = n;
        age = a;
    }

    void display() {
        System.out.println(name + " is " + age + " years old.");
    }
}

public class Main {
    public static void main(String[] args) {
        Student s1 = new Student("Aaryan", 20);
        Student s2 = new Student("Riya", 22);

        s1.display();
        s2.display();
    }
}

‚úÖ Output:

Aaryan is 20 years old.
Riya is 22 years old.

üí° Constructors are like automatic setup tools ‚Äî they give your objects values as soon as they're created.
